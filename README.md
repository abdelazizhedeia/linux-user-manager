# Linux User Manager (Python)

A simple Linux system administration tool written in Python for managing users and groups from the terminal.

## Features

- Add User
- Modify User
- Delete User
- List Users
- Add Group
- Add User to Group
- Delete Group
- List Groups
- Disable User
- Enable User
- Change User Password

## Requirements

- Linux System
- Python 3.x
- Root or sudo privileges
